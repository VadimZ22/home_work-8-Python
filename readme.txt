Все модули написаны мной самостоятельно.
Запуск программы осуществляется из модуля main
user_interface отвечает за взаимодействие с пользователем и вызывает методы из input_output.
input_output отвечает за добавление  и чтение данных в телефонную книгу (phonebook.csv) и вызывает метод записи лога.
выгрузка данных осуществляется в файлы out_file.txt или out-file.csv по указанию пользователя.
